    About validation rules, validation rule groups and validation notifications

 

A validation rule is based on an expression which defines a relationship between data element values. The expression forms a condition which should assert that certain logical criteria are met.

 

The expression consist of:

  • a left side
  • a right side
  • an operator

 

 

 

Example 9.12.

 

A validation rule could assert that the total number of vaccines given to infants is less than or equal to the total number of infants.

A validation rule group allows you to group related validation rules. When you then run a validation rule analysis, you can select whether to run all validation rules in the system, or just the validation rules in one group.

You can configure a validation rule analysis to automatically send out information about validation violations to selected user groups. These messages are called validation notifications and are sent via the internal DHIS2 messaging system.
