Create or edit a validation rule group

  1. Open the Maintenance app and click Validation > Validation rule group.

2. Click the add button.

 

Using the Maintenance appCreate or edit a validation

notification

 

  1. Type a Name.
  2. (Optional) In the Code field, assign a code.
  3. (Optional) Type a Description.
  4. Double-click the Validation rules you want to assign to the group.
  5. Click Save.
