Create or edit a data element group set

 

Data element group sets allows you to categorise multiple data element groups into a set. The system uses data element group sets during analysis and reporting to combine similar data element groups into a common theme.

  1. Open the Maintenance app and click Data elements > Data element group set.
  2. Click the add button.
  3. Fill in the form:
  1. Name
  2. Code
  3. Description
  4. Compulsory
  5. Data dimension
  1. Select data element groups and assign them.

 

Available data element groups are displayed in the left panel. Data element groups that are currently members of the data element group set are displayed in the right hand panel.

  1. Click Save.
