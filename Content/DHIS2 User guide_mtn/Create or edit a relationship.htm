Create or edit a relationship type

  1. Open the Maintenance app and click Program > Relationship type.

 

 

 

  1. Click the add button.
  2. Type a Name of the relationship type.
  3. (Optional) Assign a Code.
  4. In the A is to B field, enter a description of which relationship tracked entity A has to tracked entity B, for example mother.
  5. In the B is to A field, enter a description of which relationship tracked entity B has to tracked entity A, for example child.
  6. Click Save.
