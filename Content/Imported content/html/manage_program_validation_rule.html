26.6.1. About program validation rules

A validation rule is based on an expression which defines a relationship between data element values. The expression forms a condition which should assert that certain logical criteria are met.

The expression consist of:

  • a left side

  • a right side

  • an operator

Example 26.15. 

A validation rule could assert that an admitted date given is before the discharged date.


26.6.2. Create a program validation rule

  1. Open the Program / Attributes app and click Program.

  2. Click the program you want to add a validation rule to and select View validation rules.

    A list of existing program rules for the selected program opens. If the program doesn't have any program rules, the list is empty.

  3. Click Add new.

  4. Enter a validation rule Name.

  5. Select an Operator.

  6. Click Edit left side and create the expression’s left side.

  7. Click Edit right side and create the expression’s left side.

  8. Click Save.

26.6.3. Edit program validation rules

  1. Open the Program / Attributes app and click Program.

  2. Click the relevant program and select View validation rules.

    A list of existing validation rules for the selected program opens. If the program doesn't have any validation rules, the list is empty.

  3. Click the validation rule you want to modify and select Edit.

  4. Modify the options you want.

  5. Click Update.

26.6.4. Delete program validation rules

  1. Open the Program / Attributes app and click Program.

  2. Click the relevant program and select View validation rules.

    A list of existing validation rules for the selected program opens. If the program doesn't have any validation rules, the list is empty.

  3. Click the validation rule you want to delete and select Remove.

  4. Click OK to confirm.
